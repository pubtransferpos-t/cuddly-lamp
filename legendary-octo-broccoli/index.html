DEXE

Script Studio

v1.0.8 C++ / C# WPF + WebView2 Windows Only

Write scripts.
Run anything.

Dexe is a Lua script studio for Roblox built on C++ and C#. It writes unsigned Luau bytecode directly into Roblox core modules, giving you fast, flexible execution with multi-instance support, an HTTP spy, a virtual filesystem, and a Monaco-powered editor.

Using Dexe may result in a Roblox account ban — it is detected by Byfron. Always use an alt account. The author is not responsible for any bans or policy violations.

Features

⚡

Fast Execution

Writes unsigned Luau bytecode directly into Roblox core module scripts.

🔀

Multi-Instance

Attach to and execute across multiple Roblox instances simultaneously.

🛡️

Lua Armor

Supports executing most scripts, including Lua Armor protected ones.

💾

Virtual FS

Extremely fast virtual filesystem that syncs to external storage.

📡

HttpSpy

Monitor all HTTP requests made by scripts in real time.

🌐

Global State

Set and get globals shared across all connected clients.

Tech Stack

C++20 C# / .NET 8 WPF WebView2 Luau Compiler cpp-httplib zstd xxHash OpenSSL nlohmann/json Monaco Editor Windows API / ntdll

Custom API Examples

-- Get the real memory address of an instance local address = Dexe.get_real_address(game:GetService("ScriptContext")) print("Address:", string.format("0x%x", address)) -- Enable HTTP spy Dexe.HttpSpy() -- Share globals across all clients Dexe.SetGlobal("__test", { test_text = "hello, world!" }) local t = Dexe.GetGlobal("__test") print(t.test_text) -- hello, world! -- Spoof an instance address Dexe.spoof_instance(game:GetService("CoreGui"), 0) -- Process info print(Dexe.PID) -- current Roblox process ID print(Dexe.GUID) -- GUID used for external communication

Dependencies

  • cpp-httplib — Lightweight HTTP server for DLL ↔ UI communication
  • xxHash — Fast hashing for bytecode checksums and XOR keys
  • zstd — Compression of Roblox bytecode (RSB1 format)
  • OpenSSL — Cryptographic operations
  • nlohmann/json — JSON parsing
  • Microsoft.Web.WebView2 — Powers the Monaco-based script editor

Building

1

Build Dexe.dll

Open dll-source/Xeno.sln in Visual Studio 2022 — all source files and include paths are pre-configured. Set to Release / x64, press Ctrl+Shift+B. Output: dll-source\bin\Dexe.dll.

2

Build DexeUI.exe

Open DexeUI.cpp in Visual Studio 2022, set to Release / x64, press Ctrl+Shift+B.

3

Run

Copy dll-source\bin\Dexe.dll next to DexeUI.exe, open Roblox first, then launch the UI.
